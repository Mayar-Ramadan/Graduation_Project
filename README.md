# 💧 SipSmart – Smart Water Bottle
## 📖 Overview

**SipSmart** combines **Artificial Intelligence**, **Mobile Application Development**, and **Hardware** Integration to help users maintain healthy hydration habits.

The system **predicts** daily water consumption, **tracks** hydration in real time, and **provides** personalized **notifications** through a smart mobile application connected to an intelligent water bottle.

---

## 🎯 Objectives
- Encourage healthy hydration habits.
- Predict users' daily water intake using Machine Learning.
- Monitor water consumption in real time.
- Connect a smart water bottle with a mobile application.
- Provide personalized hydration recommendations.

## 🏗️ Project Architecture

The project is organized into three main modules:

### 🤖 AI Module
- Water consumption prediction using Machine Learning.
- CatBoost model.
- Linear Regression model.
- Dataset preprocessing and model evaluation.

### 📱 Flutter
- Cross-platform mobile application.
- User authentication.
- Hydration tracking dashboard.
- Daily reminders and notifications.

### 🔌 Hardware
- Smart water bottle prototype.
- Sensor integration.
- ESP32/Arduino communication.
- Bluetooth/Wi-Fi connectivity.
